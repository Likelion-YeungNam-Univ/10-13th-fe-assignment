## Branch Naming Convention
### ~5/25까지 코드리뷰 완료해주세요!

꼭 `main` 브랜치에서 생성해주세요!

```bash
  git checkout -b feature/{이름}
```

- 생성 후 해당 브랜치로 바로 이동
- ex. feature/kim_joo_hee

## Pull Request

- PR은 가능한 **상세하게** 써주세요!
- 코드의 경우 👇
```bash
  백팃으로 감싸서 다음과 같이 작성해주세요!
```
- 신경써서 구현한 부분은 **화면을 캡쳐**해서 소개해주셔도 됩니다!
- AI 도움을 받았다면 내용 이해 후 **자신의 언어로 재구성**하여 설명까지 해주세요!

## Code Review

- 궁금한 점, 개선점 등 상대를 위해 자세하게 써주세요!
